How to start a Visual Studio Code project

Start by creating a new folder in files, and then go to Visual Studio Code. There, you will go to the top left, and cick file, open folder, and select the folder you have created. Now go to the explorer, and click the option with a page and the plus on the page's bottom right. Set up "Index.html" "Style.css", and "main.js." Now, start coding.
